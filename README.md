CodeDrops – Your Personal Code & Note Vault

## 🌟 Features

- 📝 Create **text drops** with titles
- 📋 **Copy to clipboard** with one click
- 🔗 Generate and **share URLs** for specific drops
- ⚙️ Built using modern tech: React, Redux, Vite, TailwindCSS
- 📱 Mobile-ready layout
